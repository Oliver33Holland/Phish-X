# Phish X – Security Awareness Training Platform

Phish X is a security awareness training tool for simulating and detecting phishing emails, deepfake voices, and social engineering attacks.

---

## ⬇️ Download

**[Download Phish X (Windows)](../../releases/latest)**

Click the link above, then download `PhishX-Send.zip` from the Assets section.

---

## How to Run

1. Download and **extract** `PhishX-Send.zip`
2. Open the extracted `Phish X` folder
3. Double-click **`Phish X.exe`**

The app opens automatically in your browser at `http://127.0.0.1:8000/static/`

> No Python or installation required — everything is bundled.

---

## Features

- 48 simulated phishing emails across 16 brands
- Gamified spot-the-phish mini game
- Phishing campaign creation and tracking
- AI-powered email risk analysis
- Voice/audio deepfake vishing detector
- Fake login page simulations
- Dashboard with click rates and analytics
